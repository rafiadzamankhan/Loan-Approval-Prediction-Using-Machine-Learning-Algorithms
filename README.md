This project predicts whether a bank should approve or reject a loan application using machine learning. It was built as part of a university AI lab project.

Three models were trained and compared - Logistic Regression, Decision Tree, and Neural Network on a dataset of 45,000 applicants. The Neural Network achieved the best results with 89% accuracy and 0.97 AUC.

Key techniques used: SMOTE for class imbalance, StandardScaler for feature scaling, One-Hot and Label Encoding for categorical data, and Stratified train-test split.

Dataset features: income, credit score, loan amount, interest rate, employment history, loan intent, and more.
